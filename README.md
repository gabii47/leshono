# Leshono

Leshono is a Duolingo-style language-learning app focused on **Surayt / Turoyo**.

## Repo layout
- `apps/` — client apps (mobile/web)
- `content/` — scraped + generated course content
- `scripts/` — content pipeline tools
- `docs/` — product + technical docs

## Netlify
Netlify is configured to publish a built web artifact. See the Netlify build settings in `docs/`.
